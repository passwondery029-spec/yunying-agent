## 第二章 核心理论

### 形神合一论

形神关系即身与心、生理与心理的关系，是中医心理学的根本理论。

#### 神的概念

中医学中"神"的含义已脱离鬼神信仰本义，包含两个层面：

- **广义之神**：整个人体生命活动的外在表现。凡具有呼吸、语言、饮食、排泄等生命活动者，神便寓于其中。脏腑生理功能的外在表现、显露于外的征象均属神的内涵。
- **狭义之神**：精神意识思维活动。基本范畴相当于现代心理学中的心理过程，包括：
  - **思维活动**：感知、记忆、思考、想象、判断等认知过程。《内经》以意、志、思、虑、智概括："心有所忆谓之意，意之所存谓之志，因志而存变谓之思，因思而远慕谓之虑，因虑而处物谓之智。"
  - **感觉意识**：以神、魂、魄、意、志分述，分藏于五脏，总统于心神（五脏神）。
  - **情志活动**：喜、怒、忧、思、悲、恐、惊七情。七情变化根源于脏腑气血的正常活动，属于心神的体现。《素问·阴阳应象大论》："人有五脏化五气，以生喜怒悲忧恐。"

#### 形的概念

"形"指人的形体：视之可见、触之可及之脏腑经络组织、五官九窍、四肢百骸等有形躯体，以及循行于脏腑之内的精微物质（脏腑、经络、气血、津液、精、骨、肉、筋、脉等）。

#### 形神关系三大层次

**1. 形具而神生——形为神之质**

形体是神产生的物质基础。"生之来谓之精，两精相搏谓之神。"父精母血，两精相合构成胚胎，形成生命，产生神。神的产生依附于形体而存在，无形体则无神。

- 脏腑精气是神的物质基础："血者，神气也""神者，水谷之精气也"
- 气血充盈则神旺，气血衰弱则神衰
- 望眼神可察神："目者，五脏六腑之精也……神气之所生也"

**2. 神为形之主——神对形的统摄**

神对形具有主导和统摄作用。心为君主之官，主明则下安，主暗则十二官危。神的这一主宰作用至关重要：

- 若神的主宰正常 → 脏腑经络协调、精气血津液运行正常
- 若神的主宰失常（太过或不及）→ 影响脏腑气血，造成形体衰弱
- 七情致病直接伤五脏：怒伤肝、喜伤心、悲伤肺、思伤脾、恐伤肾
- 五脏受伤进一步影响心："悲哀愁忧则心动，心动则五脏六腑皆摇"

**3. 形与神俱——形神合一的最高境界**

"形与神俱"语出《素问·上古天真论》，指生命形体与精神心理状态的高度和谐平衡，是生命活动的基本特征，也是保身长全的重要前提。

- 张景岳："形者神之体，神者形之用。"
- 形神相合 → 生机蓬勃；形神相离 → 生机不存
- "形体不敝，精神不散" → 健康长寿

#### 形神病理互影响

- **形病则神病**：脏腑功能失调 → 神志活动异常。《素问·宣明五气》："精气并于心则喜，并于肺则悲，并于肝则忧，并于脾则畏，并于肾则恐。"心病可导致神志疾患。
- **神病形亦病**：情志过激 → 五脏功能失调 → 疾病。"怒伤肝""喜伤心""悲伤肺""思伤脾""恐伤肾"。暴怒暴喜可诱发脑出血、心肌梗死等。

### 心主神明论

心是人体生命活动的主宰，是精神意识思维活动的中枢。

#### 核心论点

- **《素问·灵兰秘典论》**："心者，君主之官，神明出焉。"
- **《灵枢·邪客》**："心者，五脏六腑之大主也，精神之所舍也。"
- 心主神明的含义：心统帅和主宰人体的精神、意识、思维、情志等心理活动

#### 心主神明的表现

1. **主宰意识思维**：心接受外界信息，进行思考判断，产生意志决定
2. **主宰情志活动**：七情虽分属五脏，但总统于心。情志活动的调节离不开心
3. **主宰生命活动**：心为五脏六腑之大主，心的功能正常则脏腑协调；心主不明则十二官危

#### 心神与五脏神的关系

- 心神是总帅，五脏神是心神在不同脏腑的具体表现
- 心藏神（总）、肝藏魂、脾藏意、肺藏魄、肾藏志
- 心神受损可波及五脏，五脏病变也可影响心神
- "悲哀愁忧则心动，心动则五脏六腑皆摇"

### 五脏神志论

五脏与神志活动的对应关系：

| 脏腑 | 神志 | 正常表现 | 失衡表现 |
|------|------|---------|---------|
| 心 | 神 | 精神饱满、思维清晰 | 失眠、多梦、心悸、健忘 |
| 肝 | 魂 | 情志舒畅、谋虑周全 | 烦躁易怒、抑郁、多梦惊恐 |
| 脾 | 意 | 思维敏捷、注意力集中 | 思虑过度、健忘、倦怠 |
| 肺 | 魄 | 勇敢果断、感知灵敏 | 悲伤忧愁、胆怯、气短 |
| 肾 | 志 | 意志坚定、记忆强 | 恐惧不安、健忘、意志消沉 |

**核心逻辑**：情志异常不是"心理问题"那么简单，而是脏腑功能失调的外在表现。
- 心悸失眠 → 心神不宁
- 烦躁易怒 → 肝魂不安
- 思虑过度 → 脾意不运
- 悲伤忧愁 → 肺魄不宁
- 恐惧不安 → 肾志不坚

## 第三章 临床基础：病因病机与诊法

### 情志致病的病因

#### 1. 体质因素——生理始基

- 体质是个体在生长发育过程中形成的差异性
- "邪之所凑，其气必虚"——体质之"虚"是发病前的生理特点
- 体质决定个体对某些疾病的易感性和倾向性

#### 2. 人格因素——易感内因

不良人格通过四个途径致病：
1. **决定行为类型和生活方式**：如太阳之人好高骛远，易发心痛、胸痹
2. **影响对社会心理刺激的认知评价**：产生不同情绪反应和生理变化
3. **影响应对方式和适应能力**：应对不当加重心理应激
4. **影响人际关系**：决定社会支持的量和质

#### 3. 七情内伤——直接病因

七情是正常精神活动，只有突然、强烈或长期持久的情志刺激，超过个体调节范围，才导致疾病。七情致病特点：

**直接伤及内脏**：
- 怒伤肝、喜伤心、思伤脾、悲伤肺、恐伤肾
- 但非绝对——人体以五脏为中心，心为主宰，情志伤脏必影响及心

**影响脏腑气机**：
- 怒则气上：暴怒 → 肝气上逆 → 头痛面赤、呕血
- 喜则气缓：过喜 → 心气涣散 → 精神不集中
- 悲则气消：过悲 → 肺气耗散 → 气短乏力
- 恐则气下：恐惧 → 肾气下陷 → 二便失禁
- 惊则气乱：受惊 → 心气紊乱 → 心悸恍惚
- 思则气结：过思 → 脾气郁结 → 胸脘满闷

**情志波动可致病情变化**：
- 情志波动常使病情急剧变化，如心脏病患者暴怒可致猝死

### 情志致病的病机

#### 1. 气机失常（核心病机）

情志致病首先干扰气机，使气的升降出入失常：
- 气滞：肝气郁结 → 胸胁胀痛
- 气逆：肝气上逆 → 头痛眩晕
- 气陷：恐则气下 → 腰酸下坠
- 气闭：思则气结 → 脘腹痞闷

#### 2. 脏腑失调

- 不同情志损伤相应脏腑："喜伤心""思伤脾""忧伤肺""怒伤肝""恐伤肾"
- 不良行为习惯也伤脏腑：酗酒、吸烟、暴食肥甘 → 心肝脾胃功能受损

#### 3. 伤精耗血

- 精血是心理活动的主要物质基础
- 先天禀赋不足、情志活动过激、欲求未遂 → 精血耗伤
- 精血亏损常见症状：耳鸣、眩晕、眼花、腰膝酸软、健忘、失眠

#### 4. 痰瘀内生

- 气机不利 → 津液停留 → 聚痰成饮
- 气滞日久 → 血行不畅 → 瘀血内生
- 痰瘀互结 → 阻滞清窍 → 神志异常
- 轻者：胸闷心悸、喘咳咯痰
- 重者：癫狂、痴呆、神志不清

### 诊法与辨证——对话信息收集指南

#### 望诊察心

**望神（核心）**："得神者昌，失神者亡"

- **得神**：眼神精彩内含，目光明亮，反应灵活，表情自然 → 正气未伤，预后良好
- **失神**：目光呆滞，反应迟钝，表情呆板，精神萎靡 → 精气亏损，预后不良
- **神气不足**：轻度失神，多见于虚证

**望眼神**：
- 目光炯炯 → 有神
- 目光黯淡 → 失神
- 双眼时时闭合 → 精神疲惫消沉
- 目光低垂浑浊 → 心事重重、忧郁
- 两目圆睁直视 → 暴怒
- 眉开眼笑 → 喜悦
- 无端嘻笑 → 狂证
- 时常落泪 → 郁证

**望面色**：
- 面色微黄红润光泽 → 祥和稳重的心理状态
- 面色黯淡无光 → 心情长期压抑忧郁
- 面色红润而眉开 → 兴奋喜悦
- 面色苍白 → 悲哀伤感
- 面色红绛青紫 → 极度愤怒
- 面色乍黑乍白 → 过度惊恐

#### 问诊察心

**问诊重点**（云英AI对话收集信息的关键）：
1. **情志变化**：发病前有无明显情志刺激（争吵、受惊、悲伤、忧虑等）
2. **情绪特征**：主导情绪是什么（愤怒/悲伤/恐惧/忧愁/思虑）
3. **睡眠情况**：入睡难/易醒/多梦/早醒
4. **饮食状态**：食欲增减/口味偏好/有无口苦口干
5. **二便情况**：大便干结/溏薄，小便黄赤/清长
6. **躯体症状**：胸闷/胁胀/心悸/头晕/腰酸/手足冷热
7. **性格倾向**：平素急躁/内向/多疑/胆小
8. **病程特点**：突发/渐发/反复/迁延

## 第四章 常用治疗方法

### 意疗（心理治疗）

#### 1. 顺情从欲

- **原理**：顺从患者的意念、情欲，满足患者的心理需要，以释心理病因。"从"即顺从、适应、遵循，取中医"从治"之意
- **经典依据**：
  - 《素问·阴阳应象大论》："从和欲快志于虚无之守"
  - 《灵枢·师传》："顺者，非独阴阳脉之道顺也，百姓人民皆欲顺其志。"——所从者不仅是"身"（生理规律），还包括"志"（意愿欲求）
  - 张景岳："若思虑不解而致病者，非得情舒愿遂，多难取效"
- **适用**：因正当需求被压抑导致的情志不畅（爱情、婚姻、家庭、求学、事业等基本愿望未得满足）
- **方法**：了解需求 → 评估合理性 → 在伦理道德许可前提下创造条件适当满足
- **核心原则**：
  - 饥而欲食、寒而欲暖、劳而欲息等是人类最基本的生理需要，不能硬性废止
  - 必要的生活欲望得不到满足 → 影响生理活动 → 甚至导致病变
  - 尊重、同情、体谅、迁就患者的情绪，适当满足愿望，有助于治疗
- **注意**：不是纵容，而是"合理的顺"；《灵枢·师传》："未有逆而能治之也，夫惟顺而已"

#### 2. 开导解惑

- **原理**：以语言为主要手段与患者交谈，使之明了与疾病有关的道理，主动消除心理障碍
- **经典话术**："告之以其败，语之以其善，导之以其所便，开之以其所苦"
  - 告之以其败：告知疾病危害（引起重视）
  - 语之以其善：告知良好前景（建立信心）
  - 导之以其所便：告知便利方法（降低门槛）
  - 开之以其所苦：开解痛苦心结（情感支持）
- **操作四步法（擒、纵、切入、突破）**：
  1. **擒**：获取患者信任，建立良好医患关系。耐心倾听，让患者倾诉痛苦和真情
  2. **纵**：进一步引导倾诉，了解深层心理问题，调动患者主观能动性
  3. **切入**：接触核心问题，针对病根进行分析说理
  4. **突破**：深入治疗，使外加治疗与患者内在积极因素结合
- **吴鞠通经验**："婉言以开导之，庄言以震惊之，危言以悚惧之，必使之心悦诚服，而后可以奏效如神"
- **孙思邈方法**：开创与患者"共语"法，提高"受入性"（对心理治疗的接受能力）
- **注意**：
  - 言语开导必须取得患者信任，在融洽气氛中进行
  - 语词适当慎重，注意为患者保密
  - 交谈要有的放矢、细致入微
  - 使用得当可立竿见影，使用不当可能加重病情——须三思而行

#### 3. 情志相胜

- **原理**：以一种情志克制另一种情志，源自《内经》五行相克理论
- **理论基础**：
  - 情志活动以五脏为物质基础，是各脏腑功能活动的表现
  - 《素问·阴阳应象大论》："人有五脏化五气，以生喜怒悲忧恐"
  - 五行相克：金胜木，木胜土，土胜水，水胜火，火胜金

- **五种相胜规则**：
  1. **怒伤肝，悲胜怒**（金克木）
     - 过度愤怒导致肝气上逆，头痛面赤
     - 以悲忧之情制约愤怒，告知令人悲忧之事使其气消
  2. **喜伤心，恐胜喜**（水克火）
     - 过度喜悦使心气涣散，精神不集中
     - 以恐惧心理制约过喜，"祸起仓卒之言"抑其过喜
  3. **思伤脾，怒胜思**（木克土）
     - 过度思虑导致脾气郁滞，胸脘满闷，食纳不旺
     - 以"污辱欺罔之言"激患者盛怒，冲破郁思
  4. **忧伤肺，喜胜忧**（火克金）
     - 过度忧愁使肺气耗散，咳喘短气，意志消沉
     - 设法使患者欢快喜悦而病愈
  5. **恐伤肾，思胜恐**（土克水）
     - 过度恐惧使肾气不固，二便失禁
     - 引导患者理性思考，了解恐惧对象的真相，以理智战胜恐惧

- **历史发展**：
  - 《吕氏春秋》载文挚以"怒胜思"治愈齐王，为现存最早记录
  - 金元四大家贡献最大，尤其张子和《儒门事亲》有治有论
  - 朱丹溪："人身诸病多生于郁"，创"越鞠丸"行气开郁
  - 《医方考》："情志过极，非药可愈，须以情胜"
  - 吴敬梓《儒林外史》"范进中举"即"恐胜喜"的典型

- **注意**：需要专业判断，不可自行使用；需辨证准确，用之得当

#### 4. 移精变气

- **原理**：转移注意力，改变精神状态，"治神以动其形"
- **经典依据**：
  - 《续名医类案》："失志不遂之病，非排遣性情不可。"
  - 张子和："先问其所好之事，好棋者与之棋，好乐者与之笔笛。"
  - 吴师机《理瀹骈文》："七情之病者，看书解闷，听曲消愁，有胜于服药者矣。"

- **两大方法**：

  **（1）精神转移法**：
  - 将患者的精神从疾病的中心或内心思虑的焦点转移、分散至其他方面
  - 适用于：过分关注自身不适、情志抑郁难以自拔
  - 具体方式：
    - 音乐歌舞：张子和找艺人载歌载舞转移注意力
    - 书画琴棋：以喜好之事分散关注
    - 户外活动：踏青、登高、游览
    - 社交互动：与友人交谈、参与集体活动
  - 要点：须针对不同心理特点、病情、环境灵活运用

  **（2）精神导引法**：
  - 通过呼吸吐纳锻炼配合动作，引导和控制精神意念
  - 不借助外界事物，以"导引"方法移情易性
  - 具体方式：
    - 呼吸吐纳：吹去热、呼去风、唏去烦、呵下气、呬散滞、咽解极
    - 配合意念引导：将注意力随呼吸移至身体不同部位
  - 适合云英AI的冥想引导场景

- **适用**：执念深重、反复纠结、过度关注自身不适

#### 5. 暗示诱导

- **原理**：通过间接暗示影响心理
- **方法**：语言暗示、环境暗示、行为暗示
- **注意**：必须是积极暗示，不可使用恐吓

#### 6. 习以平惊

- **原理**：通过反复接触令人恐惧的事物，逐渐消除恐惧反应
- **经典依据**：《内经》惊者平之，平者常也，使之习以为常
- **方法**：循序渐进地让患者面对恐惧源，从弱到强
- **适用**：惊恐发作、社交恐惧、特定恐惧

#### 7. 说理开导

- **经典话术**（出自《灵枢·师传》）：
  - **告之以其败**：告知疾病危害，引起重视
  - **语之以其善**：告知良好前景，建立信心
  - **导之以其所便**：告知便利方法，降低门槛
  - **开之以其所苦**：开解痛苦心结，情感支持
- **原理**：说理开导、同情安慰，改变患者的病态心理环境
- **方法**：说服、解释、鼓动、安慰、保证，做到动之以情、晓之以理、明之以法
- **注意**：循序渐进，不可操之过急；需先取得患者信任
- **清代吴鞠通经验**："凡治内伤者，必先祝由。详告以病所由来，使病人知之而不敢再犯，又必细体变风变雅，曲察劳人思妇之隐情，婉言以开导之，庄言以震惊之，危言以悚惧之，必使之心悦诚服，而后可以奏效如神。"

### 中医音乐疗法

> "音通人心，乐和阴阳"——音乐有益于身心健康，自古已有共识。东方音乐疗法源于中医整体观和阴阳五行学说。

#### 音乐疗法历史与理论

- **《乐记》**：我国最早音乐理论专著，提出"乐者乐也，琴瑟乐心；感物后动，审乐修德，乐以治心，血气以平"
- **《黄帝内经》**：将五音引入医学，与五脏、五志、五色、五时对应
- **五运六气学说**：五音健运，太少相生（太为有余，少为不足）
- **"乐者亦为药也"**：用乐如用药，亦有归经、升降浮沉、四气五味、配伍、个体差异
- **"草木无情而乐有情"**：养生中"用药不如用乐"

#### 五音处方的组成规律

**1. 中和之道——音乐处方的根本原则**
- 和谐不是混合，而是和而不同，在对立中找到平衡点
- 传统音乐强调"中和之美"，不追求强烈，非常宜于治疗
- 动静结合：动中求静，静中求动，与中医"阴中求阳，阳中求阴"本质一致

**2. 情绪调节——顺逆二法**
- **顺法——顺情从欲**：根据患者情绪状态，给予性质类同的音乐，使其得到共鸣和宣泄
  - 肝阳上亢（易怒）：先用升发调畅音乐宣泄 → 再以悲调制约愤怒（如《小胡笳》《江河水》《汉宫秋月》）
  - 阴虚阳亢（烦躁）：给柔和清润音乐滋阴潜阳（如《平沙落雁》《潇湘水云》）
  - 情绪低落：先给欢乐愉快音乐（如《花好月圆》《喜洋洋》）→ 再转入中和调养
- **逆法——情志相胜**：用性质不同的音乐制约、消除病态情志
  - 以悲制怒（金克木）：商调式悲伤色彩音乐制约愤怒
  - 以恐制喜（水克火）：深沉肃穆音乐制约过度兴奋

**3. 精神内守法——结合精神三角结构**
- **回归法**：通过古典音乐清净内在精神世界，达到"精神内守，真气从之"的状态
- **积极法**：主动运用正面意识思维，专注某一积极意象
- **认知调节法**：借助音乐意境调节认知，改变患者对事物的看法和态度

#### 音乐三分法（实用简明分类）

- **阳性曲目**：《弹琴》《归去来辞》《流水》《梅花三弄》《渔樵对答》——阳主动，主升，旋律流畅动感，养阳行气、强壮功效
- **中性曲目**：《春晓吟》《莲心》《双稚听凡》《玉树临风》——中主和，中正平和，养神和志、调和功效
- **阴性曲目**：《怕涧流录》《六祖偈言歌》《鸣泉忘机》《文王操》——阴主静，主降，旋律舒缓深沉，养阴潜阳、安神功效

#### 诗、歌、舞三层深入法

《乐记·师乙》认为音乐对情感抒发分三层，依次增强：
1. **诗**：用歌词表达志向感情（如词牌、曲牌、《水调歌头》）
2. **歌**：用旋律声音表达感情（如《高山流水》《广陵散》），歌吟疗法适用于胸闷气急、情志抑郁
3. **舞**：用动作配合诗歌旋律（如太极拳配合音乐），舞蹈运动可防治慢性肢体关节病变，观赏舞蹈用于郁病、嗜睡

### 气功疗法（冥想引导核心知识）

> 气功是调心、调息、调身融为一体的心身锻炼技能。三调缺一不可，三调统一是气功区别于一般体育锻炼的重要标志。

#### 适用范围与禁忌

- **适用**：各种神经症（患者保持自知力，自我意识完整，能实现意识状态的自我调控）
- **不适用**：重症精神病急性发作期（人格不完整，意识自我调控难以实现）
- **缓解期可用**：包括重症精神病的临床缓解期，可用气功延缓或防止复发
- **定位**：多作为辅助疗法，与药物疗法等其他疗法并用

#### 选功、教功、查功三步骤

**1. 选功**
- 两种方式：从已有功法中选择 / 针对病情自行设计功法
- 主辅搭配：主要功法为动功者，辅助选静功；反之选动功（动静结合原则）
- 提高疗效、防止偏差、增强练功趣味性

**2. 教功**
- 必要组成：讲授（操作要领为主）→ 演示（示范动作）→ 带练（营造气氛）→ 正误（纠正错误）
- 集体教为好，一二十人为宜
- 带练是关键环节

**3. 查功**
- 两个部分：查看练功是否规范 + 查看病情变化
- 如患者认真练功但病情未好转 → 重新检讨功法选择
- 引导和调动患者主动意识以增进疗效

#### 一、调心核心技巧

##### 1. 意守

- **定义**：在主观感觉上将意识移置于某一现实事物的心理操作
- **操作要点**：
  - "轻轻地放在那里"——似守非守，不刻意认识意守对象
  - 目的：排除杂念（以一念代万念）+ 诱导感受（引发相应身心效应）
- **意守对象**：
  - 身体部位：丹田（气机下降）、百会（气机上升）
  - 外界景物：远山（心怀坦荡、气机宣畅）、松树（气机凝重下沉）
- **与注意的区别**：
  - 注意是意识反映事物（指向+集中），意守是意识自身移位
  - 意守不要求认识对象，而是借助其单一性和感性特征排除杂念
- **云英AI应用**：引导用户将注意力"轻轻放在"呼吸上、身体某部位上

##### 2. 观想

- **定义**：意守想象中事物的心理操作
- **与意守区别**：意守对象实有，观想对象想象
- **操作要点**：想象 + 指向，更强调诱导感受
- **观想对象范围**：可想象的一切事物——自然场景、温暖光线、流水等
- **云英AI应用**：引导用户观想安全舒适的空间、温暖的场景

##### 3. 入静

- **定义**：意识活动逐渐平缓安静的过程和状态
- **层次**：
  1. 杂念减少——从纷繁到稀疏
  2. 杂念消失——一念代万念
  3. 忘却意守——意识融入感受
  4. 深度入静——物我两忘
- **入静的身心效应**：
  - 心率下降、呼吸变缓、肌张力降低
  - 大脑皮质处于保护性抑制状态
  - 自主神经系统由交感优势转为副交感优势
- **云英AI应用**：引导用户逐步入静，从关注呼吸开始

#### 二、调息（呼吸调控）

- **基本呼吸法**：
  - 自然呼吸：不加干预，顺其自然
  - 腹式呼吸：吸气腹部隆起，呼气腹部回缩
  - 逆腹式呼吸：吸气腹部回缩，呼气腹部隆起
- **六字诀**：
  - 吹——去热，呼——去风，唏——去烦，呵——下气，呬——散滞，咽——解极
- **云英AI应用**：引导4-7-8呼吸法、腹式呼吸

#### 三、调身（姿势动作）

- **基本姿势**：坐式、站式、卧式
- **原则**：松静自然，不僵不硬
- **云英AI应用**：建议冥想坐姿、放松身体各部位

### 三线放松法（失眠与焦虑核心功法）

将身体划分成三条线，自上而下依次放松：

**第一条线**：头部两侧 → 颈两侧 → 两肩 → 两上臂 → 两肘 → 两前臂 → 两手 → 十指尖 → 静养1-2分钟

**第二条线**：面部 → 颈前 → 胸部 → 腹部 → 两大腿前面 → 两膝 → 两小腿前 → 足背 → 足大趾端 → 静养大敦穴1-2分钟

**第三条线**：后脑 → 后颈 → 背部 → 腰部 → 大腿后面 → 小腿后面 → 足跟 → 足心 → 静养涌泉穴1-2分钟

**操作要点**：
- 呼吸与默念结合：吸气观想放松部位，呼气默想"松"
- 意想放松部位如海绵一样柔软
- 做完三条线后，意守丹田3-5分钟收功

### 药物疗法（连接合香产品）

> 中医用中药治疗心理疾病有两千多年历史经验积累，从心身整体观出发，认为情志变化必引起躯体生理病理变化，必须辨证施治用药调治。

#### 六大治疗法则与代表方剂

**1. 安神定志**
- **适用**：过度或持久的心理刺激引起的惊惕、失眠、健忘、多梦、烦躁惊狂等心神不宁之证
- **核心病机**：气血不足，五脏失养，神不守舍；或热邪、痰浊扰乱心神
- **辨证要点**：
  - 实证（惊狂善怒、烦躁不安）→ 重镇安神
  - 虚证（心悸健忘、虚烦失眠）→ 补养安神
  - 心肾不交（失眠遗精）→ 交通心肾
- **代表方剂**：
  - 朱砂安神丸：镇心安神、泻火养阴
  - 天王补心丹：滋阴养血、补心安神（阴亏血少的虚烦少寐、多梦健忘）
  - 酸枣仁汤：养血安神、清热除烦
  - 甘麦大枣汤：甘缓养心安冲（脏躁及郁证）
  - 归脾汤：补血养心、益气安神（心脾两虚）
  - 柏子养心丸：养心安神
- **常用药物**：朱砂、龙骨、牡蛎、酸枣仁、柏子仁、远志、夜交藤、合欢皮、小麦、五味子、龙眼肉

**2. 清心泻火**
- **适用**：五志过极化火所致的癫狂、谵语、躁扰不宁、狂动多怒、焦躁不安
- **代表方剂**：泻心汤、清营汤、犀角地黄汤、当归龙荟丸
- **常用药物**：大黄、龙胆草、栀子、黄连、夏枯草、生地黄、莲子心、竹叶、石膏

**3. 活血化瘀**
- **适用**：情志内伤致瘀血内阻，脉络闭塞；血行异常导致情志疾病
- **经典理论**：
  - "血有余则怒，不足则恐"（《素问·调经论》）
  - "血迷心包""败血上冲"等病理机制
  - 王清任："癫狂一证，乃气血凝滞，脑气与脏腑气不接"
- **代表方剂**：血府逐瘀汤（顽固性失眠）、桃核承气汤
- **常配伍**：远志、酸枣仁、丹参、郁金

**4. 疏肝解郁**
- **适用**：肝气郁结所致的情志不畅、胸胁胀满、善太息、抑郁
- **代表方剂**：逍遥散、柴胡疏肝散
- **与合香产品对应**：疏肝解郁香的核心药理方向

**5. 化痰开窍**
- **适用**：痰浊蒙蔽心窍所致的神志不清、癫狂、失眠
- **辨证分型**：
  - 痰湿内阻 → 温胆汤加减
  - 痰热内扰 → 黄连温胆汤
  - 痰热蒙窍 → 涤痰汤合安宫牛黄丸
  - 痰火上扰 → 生铁落饮
- **常用药物**：半夏、陈皮、胆南星、竹茹、石菖蒲、远志

**6. 通络利胆**
- **适用**：瘀阻脉络、闭塞官窍所致的情志疾病
- **代表方剂**：根据具体证型选用通络方药

#### 药物疗法的心理诱导作用

- **辨证论治的心理效应**：根据不同病症选用不同药物，给患者"此方专治我病"的心理暗示
- **方剂命名的心理暗示**：如"逍遥散"（治情志抑郁焦虑）、"安神丸"等命名本身就有心理诱导

## 第五章 常见心理疾病

### 郁证（抑郁）

**定义**：以心情抑郁、情绪不宁、胸部满闷、胁肋胀痛为主症

**病因病机**：
- 情志所伤为外因，体质素虚为内因
- 核心病机：气机郁滞 → 肝失疏泄 → 脾失健运 → 心神失养
- 病位在肝，涉及心、脾、肾

**分型**：

| 证型 | 核心表现 | 推荐方向 |
|------|---------|---------|
| 肝气郁结 | 精神抑郁、胸胁胀痛、善太息 | 疏肝解郁 |
| 气郁化火 | 急躁易怒、口苦咽干 | 清肝泻火 |
| 心脾两虚 | 多思善疑、心悸失眠 | 养心健脾 |
| 阴虚火旺 | 心烦少寐、头晕耳鸣 | 滋阴降火 |

**情志调养**：
1. 顺情从欲：允许自己有负面情绪，不要压抑
2. 移精变气：培养新兴趣，转移注意力
3. 喜胜忧：接触让自己开心的事物
4. 角调音乐疏肝解郁

**心理治疗要点**：
- 开导解惑为基础，让患者认识到情志因素的主导作用
- 移情易性：读书、听曲、养花种竹等，转移注意力
- 对于性格内向多愁者，鼓励广交朋友、参与活动

### 不寐（失眠）

**定义**：以经常不能获得正常睡眠为特征的一类病证

**病因病机**：
- 情志不遂 → 气机郁滞 → 化火扰心 → 心神不宁
- 思虑过度 → 心脾两虚 → 气血不足 → 心神失养
- 饮食不节 → 痰热内生 → 扰动心神
- 体虚久病 → 阴阳失调 → 阳不入阴

**分型与表现**：

| 证型 | 核心表现 | 舌脉 | 推荐方向 |
|------|---------|------|---------|
| 心脾两虚 | 多梦易醒、心悸健忘、食少倦怠 | 舌淡脉细 | 养心健脾 |
| 心肾不交 | 心烦不寐、头晕耳鸣、腰膝酸软 | 舌红脉细数 | 交通心肾 |
| 肝火扰心 | 急躁不寐、口苦目赤 | 舌红脉弦 | 清肝安神 |
| 痰热扰心 | 胸闷不寐、痰多口苦 | 舌红苔黄腻 | 清化痰热 |
| 心胆气虚 | 惊恐不寐、善惊易醒 | 舌淡脉弦细 | 益气镇惊 |

**非药物调养建议**：
1. 睡前1小时停止用手机
2. 睡前4-7-8呼吸法（吸气4秒→屏息7秒→呼气8秒）
3. 睡前温水泡脚15分钟
4. 卧室保持安静、温度适宜
5. 睡前可聆听角调/徵调音乐

**失眠专项心理疗法**：

1. **清静养神**："将一切营为计虑，举念即除，渐除渐少，渐少渐无"——减少睡前思虑
2. **移情易性**：培养兴趣爱好，转移对失眠的过度关注
3. **放松情绪法**：失眠本身的危害远不如对失眠恐惧与忧虑所造成的危害大。恐惧→紧张→失眠加重→恐惧加重→恶性循环。放松情绪、冷静接受现实至关重要
4. **松笑导眠法**：平心静气，面带微笑，行6次深而慢的呼吸后，依次意守身体各部位，每呼气时默念"松"并体会放松感觉
5. **逆向导眠法**：就寝后不去准备入睡，而是舒坦地躺着想愉快事件，或接着杂念编故事，使大脑兴奋疲劳后转入保护性抑制
6. **紧松摇头法**：仰卧后先四肢收缩用劲10秒后放松，待彻底放松后微闭双眼，头部左右缓慢摇摆，体会身体越来越松散，仿佛婴儿睡在摇篮中

### 卑惵（恐惧症/焦虑症）

**定义**：以自责无能、常怀愧意、惊恐疑惧、独居避人为主症的情志疾病

**病因病机**：
1. **心血不足**：素体阴血不足或久病耗损 → 心神失养 → 神气虚怯
2. **心胆气虚**：暴受惊骇或素体心虚胆怯 → 心神失守、胆失决断
3. **心肾阳虚**：心肾阳虚 → 神志失养 → 神机不振
4. **痰浊内阻**：肝气郁结日久 → 气机逆乱 → 郁而生痰 → 痰阻神舍

**辨证要点**：
- **辨轻重**：轻者仅情志变化时发病，自责愧意、退缩避人；重者病程可达数年，精神惶惑、惊恐疑惧
- **辨虚实**：虚证居多（心血不足、心脾两虚、心肾阳虚）；亦有虚实夹杂（痰瘀互结）

**心理治疗**：
1. **思胜恐**（情志相胜）：针对恐惧心理产生的原因，采取诱导方式，开周其思，促其广见闻
2. **说理开导**：解释恐惧的性质和原因，用肯定解答使患者心理满意
3. **系统脱敏**：鼓励患者逐步接触恐惧事物，逐渐增加刺激强度
4. **自我暗示**：让患者想象自己处在恐怖对象面前，逐渐消除恐怖心理

**气功治疗**：易筋经"韦驮献杆势"+"横担降魔杆"、八段锦"双手攀足固肾腰"+"攒拳怒目增气力"

### 脏躁（更年期综合征/癔症）

**定义**：因情志不舒或阴血亏虚，致心神不宁的情志疾病。以精神抑郁、情志烦乱、无故悲伤欲哭、或嘻笑无常、呵欠频作为主要表现

**病因病机**：
1. **先天禀赋不足**：素体阴血亏虚，五脏失养，遇情志刺激即发
2. **产后病后阴血亏虚**：多产伤血或产后病后 → 心血亏虚 → 心神不安
3. **忧思劳倦损伤心脾**：忧愁思虑伤心，劳倦过度伤脾 → 气血不足 → 神不守舍
4. **情志所伤心失所养**：所愿不遂 → 肝气郁结 → 化火扰动心神

**病位在心**，虚证为主（心血亏虚、肝肾阴虚、心脾两虚致心神失养），实证为辅（肝郁化火、痰气内扰致心神被扰）

**分型论治**：

| 证型 | 核心表现 | 推荐方向 |
|------|---------|---------|
| 心血亏虚 | 悲伤欲哭、嘻笑无常、心中烦乱、少寐多梦 | 甘润缓急、养心安神 |
| 心脾两虚 | 多虑善思、胸闷气短、失眠健忘、面色萎黄 | 补益心脾、宁心安神 |
| 心肾不交 | 心烦不寐、头晕耳鸣、腰膝酸软、月经渐少 | 滋阴降火、交通心肾 |
| 肝肾不足 | 嘻笑无常、呵欠频作、头晕耳鸣、手足心热 | 滋养肝肾、安神定志 |
| 肝郁化火 | 烦躁不宁、悲伤易哭、胸胁胀痛、口苦 | 疏肝解郁、清肝泻火 |
| 痰气内扰 | 烦躁不宁、悲喜无常、胸中窒闷、苔白厚腻 | 清热化痰、宁心安神 |

**心理治疗**：
1. **开导解惑**：针对病因、病理耐心解释，强调情志因素的主导作用，弱化病理后果，增强康复信心
2. **暗示诱导**：指导趋向心神宁静、情绪愉悦的自我暗示，注意体会暗示后的心理感受
3. **集体治疗与个体治疗结合**：通过说服、教育、保证等方式提高认知水平

### 百合病（抑郁性神经症/神经衰弱）

**定义**：因神志不遂或热病之后心肺阴虚，以精神忧侈、欲卧不能卧、欲行不能行、食欲时好时坏、口苦、尿黄、脉微数为主症

**病因病机**：
1. **七情内伤**：多愁善感、沉默寡言的性格基础，忧思过度 → 郁而化热 → 消烁心肺
2. **大病久病后**：心肺气阴耗损 → 阴虚火旺
3. **误治伤阴**：汗吐下用之失当 → 阴液重伤

**病位在心肺**，兼及肝脾。心肺阴虚 → 百脉失养 → 百合病乃成

**分型论治**：

| 证型 | 核心表现 | 推荐方向 |
|------|---------|---------|
| 心肺阴虚 | 欲食不能食、欲卧不能卧、口苦咽干、小便短赤 | 清心润肺、养阴除热 |
| 痰热内扰 | 精神恍惚、心烦失眠、口苦苔腻 | 清热化痰、养阴安神 |
| 心肺气虚 | 神疲乏力、气短懒言、心悸自汗 | 补益心肺、安神定志 |

**心理治疗**：
1. **顺势调神、畅神怡情**：保持心境恬静愉快，心胸开阔，恬淡虚无，减轻思想负担。广交朋友，花种竹，琴棋书画
2. **开导解惑**：使患者释疑解虑，正确对待周围事物和自身疾病
3. **暗示诱导**：利用简短有力语言，结合安慰剂，调整患者注意

### 梅核气（咽部异物感/神经症）

**定义**：因情志内伤、肝郁脾虚所致，以咽喉不红不肿、自觉有物梗阻、状如梅核、咯之不出、咽之不下、无碍饮食为特征

**病因病机**：
1. **肝气郁结**（核心）：情志所伤 → 肝失条达 → 气郁气滞 → 循经上结咽喉
2. **肺脾两伤**：思虑过度 → 脾失健运 → 聚湿生痰 → 痰气互结咽喉
3. **正气虚弱**：肝郁化火伤阴，或久服行气化痰药 → 损伤肺脾之阴

**气郁痰凝为基本病机**，初起多实（肝郁为主），病久由实转虚

**分型论治**：

| 证型 | 核心表现 | 推荐方向 |
|------|---------|---------|
| 肝气郁结 | 咽中异物感、精神抑郁、胸闷胁胀、善太息 | 疏肝理气、散结化痰 |
| 脾虚痰聚 | 咽中异物感、食欲不振、倦怠乏力 | 健脾理气化痰 |

**心理治疗**：
1. **怡情畅神**：引导保持心境恬静，参加有趣文体活动，转移对疾病的注意力。读书、听曲、浇花种竹等
2. **行为矫正**：帮助学会自我观察性格特征，避免情志过极
3. **暗示诱导**：通过仔细检查使患者相信无严重疾病
4. **松弛疗法**：三节放松操——深呼吸→握拳放松→抬臂放松

### 心悸

**定义**：以心中悸动、惊惕不安为主症

**手串数据关联**：
- 心率突然升高 + 静息状态 → 疑似心悸
- HRV 持续偏低 → 心神不宁信号

**分型**：

| 证型 | 核心表现 | 手串信号 |

**注意**：心悸严重或持续不缓解时，应建议用户就医检查，排除器质性心脏病

## 第六章 中医心理养生

### 养生原则

#### 1. 顺应自然

- **核心**：人的生活起居及情志活动应顺应自然界运动变化规律
- **经典依据**：
  - 《灵枢·本神》："智者之养生也，必顺四时而适寒暑，和喜怒而安居处，节阴阳而调刚柔，如是则僻邪不至，长生久视。"
  - 《灵枢·岁露论》："人与天地相参也，与日月相应也。"
- **两方面内容**：
  - 遵循自然界正常变化规律
  - 慎防异常自然变化的影响
- **主动性**：不是被动适应，而是主动调节心神以防御外邪
- **四时调神**（《素问·四气调神大论》）：
  - 春：使志生——精神活跃、志向萌发
  - 夏：使志无怒——精神充沛、不发怒
  - 秋：使志安宁——精神内敛、安宁平静
  - 冬：使志若伏若匿——精神潜藏、不外露

#### 2. 形与神俱

- **核心**：生理健康与心理健康统一，养生以养神为要
- **经典依据**：
  - "太上养神，其次养形"
  - 《素问·上古天真论》："故能形与神俱，而尽终其天年，度百岁乃去。"
- **心理健康标准**：
  - 心理活动（认知、情感、行为、个性）协调统一
  - 心理与环境协调统一
  - 心理与躯体功能协调统一

#### 3. 动静合一

- **核心**：动以养形，静以养神，动静结合
- 过度运动耗气，过度静止郁滞，须适度

#### 4. 神情相应

- **核心**：情志活动与精神状态相互影响
- 调情志即调神，养神即养情

#### 5. 审因制宜

- **核心**：根据个体差异、环境变化调整养生方法
- 因人、因时、因地制宜

### 常用心理养生方法

#### 1. 清静养神法

- **核心**："心常清静则神安，神安则七神皆安，以此养生则寿"
- **机制**："静则神藏，躁则消亡"——内心清静，精气神日益充沛；心浮气躁，精神不断衰弱
- **脑科学研究**：内心平静时脑电图出现均匀持续的α波，大脑皮质趋向主动性内抑制，有利于神经系统功能调整
- **三大操作**：
  - **守神**（精神内守）：对自己的意识思维进行自我锻炼、自我控制、自我调节，使精神专一不致外越。"嗜欲不能劳其目，淫邪不能惑其心"
  - **御神**（驾驭心神）：有意识地控制和调节心神，既不使情志太过，又不使其持久，做驾驭自己情感的主人。"养生在凝神，神凝则气聚，气聚则形全"
  - **放松**：身体和精神紧张状态的解除。交感神经活动水平下降，消除主观焦虑，提高应对应激能力。呼吸频率和心率减慢，血压下降，四肢温暖

#### 2. 养性调神法（修身养性）

- **核心**：道德修养与心理健康密切相关。"仁者寿""有大德必得其寿"
- **具体方法**：
  - **心存善良**：以他人之乐为乐，心中常有欣慰、愉悦、轻松之感，保持泰然自若的心理状态
  - **宽容忍让**：严于律己、宽以待人，避免心理紧张。不会宽容的人往往处于紧张状态，导致神经兴奋、血管收缩、血压升高
  - **节制欲望**："恬淡虚无，真气从之，精神内守，病安从来"
  - **知足常乐**："以恬愉为务，以自得为功"——哈佛大学60年研究显示，75%活到75岁者多是知足常乐的人
- **孔子三戒**："少之时，血气未定，戒之在色；及其壮也，血气方刚，戒之在斗；及其老也，血气既衰，戒之在得。"
- **养心八珍汤**（洪昭光）：慈爱心一片 + 好心肠二寸 + 正气三分 + 宽容四钱 + 孝顺常想 + 老实适量 + 奉献不拘 + 回报不求

#### 3. 节欲守神法

- **核心**："法于阴阳，和于术数，食饮有节，起居有常，不妄作劳，故能形与神俱"
- **要点**：
  - 节房劳、保阴精
  - 戒醇酒、保精气
  - "恬淡虚无"——思想娴静，没有过分欲望
  - 知足——在条件许可范围内得到相应享受，不增加思想负担

#### 4. 怡情畅神法

- **核心**："喜则气和志达，营卫通利"——心情愉快使人体营卫气血运行正常
- **方法**：
  - 寄情山水："知者乐水，仁者乐山"
  - 琴棋书画：章斋十乐——读义理书、学法帖字、澄心静坐、益友清谈、小酌半醺、养花种竹、听琴玩鹤、焚香煎茶、登城观山、寓意弈棋
  - 日常八乐：静坐之乐、读书之乐、赏花之乐、玩月之乐、观画之乐、听鸟之乐、狂歌之乐、高卧之乐
  - "笑一笑，少一少；恼一恼，老一老"
- **注意**：乐不可过激——"大喜破阴" "人但知过悲过哀足以害性，而不知过喜过乐亦足以伤生"。心理健康者能协调处理好乐中有忧的辩证关系

#### 5. 顺时调神法（四时调神）

- **核心**："智者之养生也，顺四时而适寒暑"
- **人与自然统一**："人以天地之气生，四时之法成"——人必须顺应自然界规律才能健康长寿
- **四季调神要点**：
  - 春三月"夜卧早起，广步于庭"——精神活跃
  - 夏三月"夜卧早起，无厌于日"——精神充沛
  - 秋三月"早卧早起，与鸡俱兴"——精神内敛
  - 冬三月"早卧晚起，必待日光"——精神潜藏

#### 6. 调摄情志法

- **核心**："卫身先卫心，护形先护神"
- **方法**：
  - 节制法：适度控制情志，不过度放纵喜怒
  - 疏泄法：合理表达情绪，适当宣泄
  - 转移法：以新兴趣替代旧执念
  - 以情胜情法：运用情志相胜原理

#### 7. 起居有常法

- 规律作息，不过劳不贪睡
- 顺应自然节律，日出而作日落而息
- 睡眠为养生第一要务

#### 8. 饮食调养法

- 食饮有节，不偏不倚
- 五味入五脏：酸入肝、苦入心、甘入脾、辛入肺、咸入肾
- 不可偏嗜一味，否则伤及对应脏腑
